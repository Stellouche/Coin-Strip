Bienvenue sur Coin-Strip

Règle du jeu Coin-Strip

Coin-Strip est un jeu de stratégie pour deux joueurs. Au début du jeu, les pièces sont réparties de manière alléatoire.
Le but du jeu est d’empêcher l'adversaire de jouer.Les pièces peuvent être déplacées seulement vers la gauche, sur les cases libres
Le joueur qui déplace la dernière pièce qaund elle sont toute allignées à gauche gagne la partie.

Vous pouvez modifier le jeu (nombre de case et de pièce) dans le code Javascript à l’endroit //début du programme ;)

- F5 pour recommencer la partie ou changer l'ordre des pièces
- Clicker sur la case où vous voulez déplacer
- Il doit forcement avoir une pièce à droite de là où vous choisisser d'aller

Bonne chance et bon jeu !
